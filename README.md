# Calculator for Engineers
A command-line based calculator with arithmetic, exponentiation, logarithmic, trigonometric, and conversion functionalities.

---

## Features
#### Basic Arithmetic
* Addition
* Subtraction
* Multiplication
* Division

#### Exponentiation
* Powers
* Square Root
* Cube Root

#### Logarithm 
* Base 10
* Natural
* Any Base

#### Trigonometry (with Inverse)
* Sine
* Cosine
* Tangent

#### Number System Conversion 
* Decimal
* Binary
* Octal
* Hexadecimal

#### Others
* Factorial
* Fibonacci
* Primality Test
* Set Numeric Scale
